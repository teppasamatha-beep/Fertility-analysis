# Fertility-analysis
This project presents an interactive Fertility Dataset Dashboard built to analyze lifestyle and health factors affecting fertility outcomes. The dashboard provides clear visual insights into age distribution, smoking habits, alcohol consumption frequency, trauma history, and diagnosis categories (Normal vs Altered).
Objective is to o explore how lifestyle and medical factors correlate with fertility diagnosis and provide data-driven insights through visualization.


Key Features:
1. Average age analysis of individuals.
2. Smoking habit distribution by age.
3. Alcohol consumption frequency breakdown.
4. Accident/serious trauma impact comparison.
5. Diagnosis-wise comparison (Normal vs Altered).
6. Combined analysis of smoking and diagnosis.


Insights Derived:
1. Majority of individuals are around 30 years old.
2. Smoking and alcohol usage are common in reproductive-age adults.
3. Diagnosis distribution is relatively balanced.
4. Lifestyle factors show potential influence on fertility outcomes.

Tools Used: Power BI
